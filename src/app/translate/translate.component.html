Übersetzen

Auf den ersten Blick scheint es einfach zwischen verschiedenen Sprachen zu Übersetzen. Für einzelne Wörter ist das oft auch korrekt. Sobald wir jedoch komplexe Sätz übersetzen wollen, wird es schnell sehr schwierig, da verschiedene Sprachen auch verschiedene Grammatiken voraussetzen, die sehr kompliziert sein können.

Aus diesem Grund wird für alle gängigen Online-Dienste, wie zum Beipspiel Google Translate, die Anfrage eines Nutzer, der etwas übersetzen möchte, an einen Server gesendet, welcher dann diese komplexe Berechnung ausführt.

Ein Problem mit deiser Herangehensweise ist die Privatsphäre des Nutzers, die durch die Übermittlung aller Anfragen an einen entfernten Server natürlich beeinträchtigt werden kann. Wäre es nicht toll wenn es eine Technologie gäbe, die es uns ermöglicht im Browser ein komplexes Modell auszuführen, welches es uns ermöglicht beliebige Sätze zu übersetzen.

Dank des maschinellen Lernens ist es tatsächlich heutzutage möglich genau dies zu tun. Vorausgesetzt das Gerät des Nutzers hat ausreichend Leistung, kann einfach ein trainiertes Modell heruntergeladen werden, womit beliebige Sätze übersetzt werden können ohne, dass private Daten an fremde Server übertragen werden.
