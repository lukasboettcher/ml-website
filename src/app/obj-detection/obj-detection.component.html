Objekterkennung

Diese Komponente verwendet ein vortrainiertes Modell namens Coco Single Shot MultiBox Detection, kurz cocoSSD. CocoSSD ist ein Modell, welches mittels der Comon Objects in Context Datenbank trainiert wurde. Diese Datenbank beinhaltet ca. 330 tausend Bildern aus über 170 Kategorien, welche durch das Modell zugeordnet werden können.

Dadurch, dass das Modell ziemlich kompakt gehalten ist, kann es auf so ziemlich jedem Endgerät verwendet werden. Dadurch ergeben sich eine Vielzahl von Anwendungsmöglichkeiten - von Überwachungskameras, bis zu Fotofiltern in diversen sozialen Netzwerken.

Natürlich kann ein solches Modell auch durch eigene Klassifizierungen ergänzt werden, wie das funktioniert findest du in der Klassifizierungs Komponente heraus.
