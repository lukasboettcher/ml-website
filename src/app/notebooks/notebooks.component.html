Notebooks!

Wenn du dich mit den Themen auf dieser Steite vertraut gemacht hast, und gerne einige der Konzepte in der Praxis umsetzen möchtest, bieten wir hier einige Jupyter Notebooks an, worin du in der Programmiersprache Python das Erlernte umsetzen kannst.

Hierfür ist es empfohlen, dass du dich im Voraus mit der Sprache Python vertraut machst, um die Grundlagen zu verstehen.

Die Notebooks können verwendet werden, ohne etwas herunterzuladen. Hierfür verwenden wir den Service mybinder.org. Wenn viele Nutzer zur gleichen Zeit versuchen diesen Service zu verwenden, kann es vorkommen, dass nicht genug Ressourcen für alle Nutzer verfügbar sind. In diesem Fall kannst du trotzdem die Notebooks auf nbviewer.org anschauen, ohne sie auszuführen. Alternativ kannst du die Notebooks von unserem GitHub Repository herunterladen und auf deinem Computer selber ausführen wenn du mit Jupyter Notebooks schon vertraut bist.
